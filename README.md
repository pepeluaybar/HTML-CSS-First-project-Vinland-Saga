# HTML-CSS-First-proyect

Proyecto web inspirado en **Vinland Saga**, construido con **HTML** y **CSS** como ejercicio de maquetación, semántica y orden visual. La meta es mostrar una base sólida de front‑end, con enfoque en estructura limpia, consistencia y atención al detalle.

## ¿Qué encontrarás en este repositorio?

- **Maquetación en HTML semántico**: estructura clara y entendible que facilita mantenimiento y escalabilidad.
- **Estilos en CSS**: diseño visual coherente con jerarquía tipográfica y distribución cuidada.
- **Organización del proyecto**: separación de responsabilidades para que el código sea legible y fácil de continuar.

## Objetivo del proyecto

Este proyecto nace para practicar y demostrar habilidades de **maquetación estática**. Está pensado como una base que puede seguir creciendo con más secciones, ajustes visuales y mejoras. Cada detalle se trabaja con paciencia para reflejar dedicación en el resultado final.

## Compromiso y dedicación

Este repositorio refleja el esfuerzo de construir una página desde cero, cuidando:

- **Claridad en el código** (fácil de leer y mantener).
- **Consistencia visual** (mismos criterios de color, espacios y tipografía).
- **Atención al detalle** (alineaciones, separación de bloques y orden lógico).

La intención es que cualquier empresa pueda ver no solo el resultado final, sino el cuidado con el que está hecho.

## Tecnologías usadas

- **HTML5**
- **CSS3**
